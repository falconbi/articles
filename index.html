Open source

Tools & Apps

Open-source tools built for TM1/Planning Analytics environments. Clone, adapt, use freely.

  • tm1_apportionment
    Activity Based Cost Apportionment engine. Full five-stage model with Python reciprocal solver, validation gate, and reconciliation checks. Deploy to TM1 entirely from code.
    TM1 github ↗
  • tm1_cubemap
    Interactive data lineage diagram for TM1. Visualise cube-to-cube data flow, identify upstream dependencies, and explore your model's full topology.
    TM1 github ↗
  • tm1_report_writer
    Governed financial report pack builder for TM1. Produces consistent, repeatable report sets from structured view definitions. Includes a worked Falcon Airways sample.
    TM1 github ↗
  • tm1_paw_tree
    PAW governance and visibility tool. Browse and export your Planning Analytics Workspace tree, identify orphaned assets, and audit what's published.
    TM1 github ↗
  • tm1_ide
    Visual MDX editor and TI development environment. Write, test, and deploy TurboIntegrator processes with syntax support and in-place execution against a live server.
    TM1 github ↗
  • tm1_period_dimension
    Period dimension builder for TM1. Generates consistent year/month/quarter hierarchies from a single config file. Handles fiscal year offsets and aliased display names.
    TM1 github ↗
